Charakterystyka bramki XOR

Bramka XOR, znana również jako bramka wykluczająca, to bramka logiczna, która zwraca true tylko wtedy, gdy dokładnie jedno z jej wejść jest true.

Równanie

Równanie dla bramki XOR może być przedstawione jako: Q = A XOR B.

Zastosowanie

Bramka XOR jest kluczowa w operacjach takich jak sumowanie binarne, kryptografia i systemy detekcji błędów.
